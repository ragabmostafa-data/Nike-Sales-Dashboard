# 👟 Nike Sales Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard built to analyze Nike sales performance across regions, product lines, customer segments, and sales channels.

The dashboard provides business insights that help decision-makers monitor revenue, profit, orders, and customer behavior.

## 🛠 Tools Used

---

## 📈 KPIs

- Total Revenue
- Total Profit
- Total Orders
- Total Units Sold
- Average Order Value
- Average Discount

---

## 📊 Dashboard Features

- Revenue Trend Analysis
- Sales Channel Analysis
- Revenue by Region
- Revenue by Product Line
- Revenue by Gender
- Units Sold Analysis
- Interactive Filters
- Executive Summary Cards

---

## 📌 Key Business Insights

- Kolkata generated the highest revenue.
- Online was the best-performing sales channel.
- Training was the top-performing product line.
- Men generated the highest revenue contribution.
- Average Order Value reached 288.
- Average Discount was 21%.
